# Вебдодаток попіксельного порівняння зображень (Vite)

Клієнтський вебзастосунок для **порівняння двох зображень піксель-до-пікселя** з кількома режимами візуалізації (підсвітка, теплова карта, абсолютні різниці RGB) і метриками (**% різних пікселів, кількість різних пікселів, MSE, PSNR**). Уся обробка виконується **локально в браузері**.

---

## Короткий чекліст етапів розробки
1. Ініціалізувати Vite + TypeScript; створити базову структуру (`index.html`, `src/`, `vite.config.ts`).
2. Зібрати UI завантаження файлів + контролів (поріг, режим, підгонка розміру B під A).
3. Винести ядро порівняння в окремий модуль `src/lib/imageCompare.ts` (Canvas → ImageData → обчислення).
4. Додати візуалізації: **Highlight**, **Heatmap**, **ΔRGB**.
5. Додати метрики: **diffPixelsPct**, **diffPixelsCount**, **MSE**, **PSNR**.
6. Перевірити сценарії: однакові зображення, JPEG-шум, різні розміри (з/без підгонки).
7. Зібрати (`npm run build`) і розгорнути на GitHub Pages (у `vite.config.ts` налаштовано `base: './'`).

---

## Функціонал
- Завантаження **двох зображень** (A — еталон, B — для порівняння).
- Опція **«Примусово підганяти B під A»** (масштабування B до розміру A).
- **Режими відображення** різниць:
  - **Підсвітка (Highlight)** — червоним позначаються пікселі, де різниця перевищує поріг.
  - **Теплова карта (Heatmap)** — від синього до червоного залежно від сили відмінностей.
  - **Абсолютні різниці (ΔRGB)** — показує |ΔR|, |ΔG|, |ΔB| для кожного пікселя.
- **Метрики**:
  - `diffPixelsPct` — відсоток «інших» пікселів.
  - `diffPixelsCount` — кількість «інших» пікселів у штуках.
  - `MSE` — середньоквадратична помилка.
  - `PSNR` — відношення сигнал/шум (чим більше — тим схожіші зображення).
- Перегляд A/B: **повзунок** (overlay) і **миготіння**.
- Експорт **DIFF** у PNG.
- Охайний **плейсхолдер** у перегляді до завантаження файлів.
